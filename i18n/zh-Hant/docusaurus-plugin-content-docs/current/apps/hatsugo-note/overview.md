# Hatsu-go

Hatsu-go 是一款用來每天記錄孩子語言成長，並方便日後回顧的 App。
它的設計目標是讓你能以單字為單位累積記錄，並透過圖表與摘要清楚整理變化。

我們重視的是，讓你能輕鬆留下日常觀察，例如「什麼時候開始增加了哪些詞語」「第一次說出的話是什麼」，而不造成負擔。

## 這個 App 的目標

孩子的語言成長，是每天細微變化的累積。不過，在日常生活中，要一直準確記住這些變化並不容易。

Hatsu-go 的目的，是把這些每天的發現留下成記錄，讓成長的脈絡更容易回顧。
透過持續記錄，可以把「好像詞語變多了」轉化為「這段時期開始增加了這些詞語」來回顧。

## 基本設計

Hatsu-go 同時重視記錄的容易度與回顧的方便性。

記錄時，可以為每個單字留下日期與發語狀態。也可以依需要加入備註與影片，更具體地保存當時的情境。
回顧時，則透過清單、歷史、圖表與摘要，讓個別記錄與整體變化都更容易掌握。

它不是單純的備忘錄，而是以「留下的記錄能在之後派上用場」為核心來設計。

## 記錄功能

在 Hatsu-go 中，可以把每天的發語依單字整理並記錄。
由於能同時保存發語狀態與日期，因此更容易掌握哪些詞語大約從什麼時候開始被使用。

也可以加入任意備註，留下當時的情境與印象。
像是「在這種場景說出來了」「最近常常使用」這樣的小補充，在之後回顧時可能會有很大的意義。

## 回顧功能

記錄的內容不只可以用清單或單字歷史查看，也可以透過圖表與摘要確認。

清單方便回顧最近登錄的內容，單字詳細頁則方便追蹤該詞語的歷史。
圖表可以用視覺方式確認已記錄詞語如何逐漸累積。
摘要則能簡短整理並掌握整體狀況。

這樣的設計不只是為了「留下」記錄，也希望能連結到「注意到變化」。

## 也適合發展支援情境的設計

Hatsu-go 以家庭中的記錄為中心，同時也考量到正在接受發展支援的家庭能方便使用。

發語記錄、圖表、摘要等基本功能都能免費且不限次數使用，保留了持續日常記錄所需要的基礎。
在重視支援時常需要的「持續記錄」「容易回顧」「變化容易看見」的同時，也整理成不過度專業、在家庭中容易持續使用的形式。

## 記錄的樂趣

Hatsu-go 也重視不要讓記錄變成一種義務。
留下第一次說出的詞語，或回顧詞彙一點一點增加的樣子，會成為日常育兒中的一大喜悅。

一句不經意的話，日後回顧時也可能成為重要的成長記錄。
Hatsu-go 希望支持你累積這些小小「說出來了」的樂趣。

## Basic 與 Premium 的想法

Hatsu-go 明確區分 Basic 與 Premium 的角色。

Basic 可以免費且不限次數使用發語記錄、圖表、摘要等基本記錄功能。
我們重視的是，為想先開始記錄、想持續留下日常變化的人，提供足夠實用的基礎。

Premium 則會解除影片保存上限，並可使用備份功能。
這是為了想不只保存詞語，也想更豐富地留下發語時的樣子與互動氛圍，或想集中管理重要記錄的人所準備的擴充功能。

## 安心使用

記錄資料基本上是在裝置內處理。
備份功能與分享功能也設計成使用者在需要時可以自行使用。

孩子的記錄是非常個人且重要的資料。
因此，Hatsu-go 重視不要讓使用方式過度複雜，而是以清楚、容易處理、能持續使用的形式提供。

## Hatsu-go 重視的事

Hatsu-go 重視的不是增加記錄本身，而是透過記錄讓成長更容易回顧。

讓日常的語言成長能自然、溫和、持續地留下來。
並且在之後回顧時，能覺得「幸好有留下來」。
Hatsu-go 就是為了這件事而存在的記錄 App。
