# {{アプリ名}} 外向け文言

ストア掲載、スクリーンショット、広告文、Web 掲載で使う表現の安全ラインと、
各ストアの申請項目をまとめる。外向けには誇張・断定・医学的効果を主張しない。

このページは共通の安全ラインを定義し、実際の申請文言は各ストアのページに置く。

| [審査・スクリーンショット・プライバシー](./review-notes.md) | 審査メモ、スクショ文言、データ安全性の申告 |

初版の対応言語（例: 日本語のみ）を書く。多言語対応時は本ディレクトリに
`appstore-en.md` などを追加して言語をそろえる。

## 基本方針

- このアプリを一言でどう説明するかを書く（例: 「◯◯を支える△△アプリ」）。
- 誇張・断定を避け、支援・補助の表現に寄せる（例: 「◯◯できる」ではなく「◯◯を支える」）。
- 収集していないデータ・使っていないセンサーを、使っているように見せない。
- 主語をユーザーに置き、提供価値を具体的に伝える。
- 課金訴求の範囲を明示し、基本機能を有料で解放するように見せない。
