## 審査メモ（Review Notes）に入れる前提

- アプリの種別（例: 医療アプリではない 等、誤解を避ける前提）。
- 使うセンサー・権限とその理由。使わないもの（マイク・位置情報など）も明記する。
- バックグラウンド動作・通知権限を使う場合はその必要性。
- 課金の範囲（無料機能と有料機能の境界、サブスクの有無）。
- アカウント登録・ログインの有無、オフライン動作の可否。
