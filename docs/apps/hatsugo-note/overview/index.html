発語ノート

発語ノートは、お子さまの発語記録や動画、成長の記録を、主に端末内で管理できるアプリです。

できること​

  • 発語記録の作成、編集、削除
  • 動画の撮影、取り込み、再生
  • 発語の成長グラフや統計の表示
  • 名前、性別、生年月日、写真、言語設定の管理
  • 記録サマリー画像の共有
  • Premium による動画保存数の拡張と暗号化バックアップ
