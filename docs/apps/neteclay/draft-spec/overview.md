# NeteClay 仕様

NeteClay は、子どもの**寝かしつけ**のための音楽アプリ。抱っこ・トントンに合う比較的速めのテンポで
始まり、時間をかけてゆっくりになる。眠りが深くなるまでの時間帯を低い音で支え、生活音でのぶり返しを防ぎ、
そのあと静かに消える。サーバーを使わず、すべて端末内で完結する。iOS / Android 両対応。

## 設計の芯: 父親が楽をする

疲れて子を抱いた親が、半分眠りながらでも使えることを最優先にする。
既定パスは、内蔵の既定値、ホームのワンタップ、放置で自動停止まで進むゼロ操作の流れにする。
タイムライン調整や再生中のタップは、使いたい親だけの任意操作とする。
詳細は[コンセプト](./concept.md)を正とする。
