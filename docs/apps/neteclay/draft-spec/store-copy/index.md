# NeteClay 外向け文言

ストア掲載、スクリーンショット、広告文、Web 掲載で使う表現の安全ラインと、
各ストアの申請項目をまとめる。仕様本文では説明のために睡眠状態を扱うが、
外向けには医学的・治療的な効果を主張しない。

このページは共通の安全ラインを定義し、実際の申請文言は各ストアのページに置く。

## 基本方針

- 「寝かしつけを支える音楽アプリ」として説明する。
- 「睡眠を改善する」「眠れるようになる」ではなく、「起きにくい環境を支える」「寝かしつけの流れに寄り添う」と表現する。
- マイク・センサーで睡眠を検出するような印象を与えない。
- 子どもの状態を診断・測定する表現を使わない。
- 主語は親に置く。「親が少ない操作で寝かしつけを始められる」ことを伝える。
- 課金訴求は音色の追加に限定する。寝つき機能を有料で解放するように見せない。

## 使ってよい表現

- 抱っこやトントンに合うテンポで始まる。
- 時間とともに音がゆっくり落ち着く。
- 寝ついたあとも低い音で見守る。
- 眠りが深くなるころ、静かにフェードアウトする。
- 生活音が気になりにくい環境を支える。
- マイクやセンサーを使わず、端末内で完結する。
- ホームのワンタップで始められる。

## 避ける表現

- 必ず寝る。
- 夜泣きが治る。
- 睡眠の質が改善する。
- 睡眠障害を改善する。
- 医師監修でないのに医学的効果を示す。
- 睡眠ステージを検出する。
- 子どもの眠りを測定する。
- 科学的に証明された効果がある。

## アプリ名の扱い

アプリ名の正式表記は[決定ログ](../decisions.md)で未決。ストア名・商標の確認が済むまで、
各ストアページの App Name / タイトルは暫定表記として扱い、確定後に全ページを同じ表記へそろえる。
