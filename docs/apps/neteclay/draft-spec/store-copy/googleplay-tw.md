## 完整說明

```
<b>【從懷抱的節奏，到安靜的入睡】</b>
NeteClay 是一款陪伴哄睡的音樂 App。
音樂以貼近抱哄與輕拍、稍快一些的節奏開始，再隨時間逐漸放慢。
孩子入睡後，會以低而安靜的聲音守候；等睡得更沉時便淡出並自動停止。

<b>【為了讓疲累的父母輕鬆一些】</b>
只要在主畫面按一次大按鈕，之後便自動進行並自動停止。
不需要輸入月齡、不需調整音量、也不必挑選音色。細部調整只是想用時的選項。

<b>【主要特色】</b>
✅ <b>一鍵開始</b>（單手抱著孩子也好按的大型播放按鈕）
✅ <b>聲音逐漸沉靜</b>（節奏隨時間放慢，深眠階段以低音支撐）
✅ <b>再哄一次</b>（孩子躁動時緩緩回到前段音樂）
✅ <b>自動／手動模式</b>（放著就結束，或按「睡著了」進入深眠）
✅ <b>時間軸調整</b>（憑感覺調整入睡與睡沉的時點；選用）
✅ <b>低刺激畫面</b>（昏暗房間也不刺眼；哄睡途中不跳彈窗）

<b>【安心使用】</b>
不使用麥克風、感測器與定位。不使用伺服器，設定與紀錄只保存在裝置內。
飛航模式下也能使用（購買與還原除外）。

<b>【關於費用】</b>
哄睡的基本功能全部免費。付費內容僅有一次買斷的音色包，沒有訂閱制。
與入睡直接相關的功能絕不以付費解鎖。

<b>【使用須知】</b>
NeteClay 以聲音支援哄睡環境，並非醫療或治療用途。
不進行睡眠測量或診斷，也不保證孩子的睡眠。
若對孩子的睡眠或發展有疑慮，請諮詢專業人士。
```
