## Description

```
【抱っこのリズムから、静かな寝入りへ】
NeteClay（ネテクレイ）は、子どもの寝かしつけに寄り添う音楽アプリです。
抱っこやトントンに合う比較的速めのテンポで音楽が始まり、時間をかけてゆっくり落ち着いていきます。
寝ついたあとも低い音でそっと見守り、眠りが深くなったころ、静かにフェードアウトして自動で止まります。
急に音量が変わらないよう、始まりも終わりもゆっくり。うとうとした子を驚かせにくい音の流れにしています。

【父親が楽をするために作りました】
疲れて子を抱いた親が、半分眠りながらでも使えることを最優先にしました。
ホームの大きなボタンをワンタップするだけで始まり、あとは放っておけば自動で止まります。
月齢入力も音量調整もプリセット選びも求めません。細かい設定は、使いたい夜だけの任意操作です。

【主な特長】
・ワンタップで開始。片手・抱っこ中でも押しやすい大きな再生ボタン。
・時間とともにテンポがゆっくりになり、深まりの時間帯は低い音で支える。
・「寝ない」で、ぐずったときにそっと導入寄りへ戻す。
・「寝た」を押して深まりへ移る手動モードと、放置で完結する自動モード。
・寝つくころ・深くなるころを、感覚的に動かせるタイムライン調整（任意）。
・暗い部屋でまぶしくない、低刺激の画面。就寝中に通知やダイアログを出しません。

【安心して使えます】
・マイク・センサー・位置情報は使いません。
・サーバーを使わず、設定や記録はすべて端末内にのみ保存します。
・機内モードでも、寝かしつけ再生・設定・記録の確認ができます。

【料金について】
寝かしつけの基本機能（テンポの変わる再生・自動停止・タイムライン調整・記録）はすべて無料です。
有料は音色の楽しみを広げる音源パックの買い切りのみで、サブスクリプションはありません。
寝つきに直結する機能を有料で解放することはありません。

【ご利用にあたって】
NeteClay は寝かしつけの環境を音で支えるアプリで、医療・治療を目的としたものではありません。
睡眠の測定・診断は行いません。子どもの睡眠そのものを保証するものではありません。
子どもの睡眠や発達について気がかりがある場合は、専門家にご相談ください。
```
