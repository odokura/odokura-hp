# NeteClay ストア入力情報：Google Play（日本語）

各項目は[概要・安全ライン](./index.md)の表現ルールに従う。文字数上限は末尾の Notes を参照。
詳細説明の `<b>` タグは Google Play の書式指定に従い、そのまま入力する。

## タイトル

```
NeteClay：寝かしつけの音楽
```

## 短い説明

```
抱っこのリズムに寄り添う音楽で、寝かしつけの流れをそっと支える。マイク不使用・端末内で完結する育児アプリ。
```

## 詳細説明

```
<b>【抱っこのリズムから、静かな寝入りへ】</b>
NeteClay（ネテクレイ）は、子どもの寝かしつけに寄り添う音楽アプリです。
抱っこやトントンに合う比較的速めのテンポで音楽が始まり、時間をかけてゆっくり落ち着いていきます。
寝ついたあとも低い音でそっと見守り、眠りが深くなったころ、静かにフェードアウトして自動で止まります。

<b>【父親が楽をするために作りました】</b>
疲れて子を抱いた親が、半分眠りながらでも使えることを最優先にしました。
ホームの大きなボタンをワンタップするだけで始まり、あとは放っておけば自動で止まります。
月齢入力も音量調整もプリセット選びも求めません。細かい設定は、使いたい夜だけの任意操作です。

<b>【主な特長】</b>
✅ <b>ワンタップで開始</b>（片手・抱っこ中でも押しやすい大きな再生ボタン）
✅ <b>ゆっくり落ち着く音</b>（時間とともにテンポが下がり、深まりの時間帯は低い音で支える）
✅ <b>寝ない</b>（ぐずったときにそっと導入寄りへ戻す）
✅ <b>自動モード / 手動モード</b>（放置で完結、または「寝た」で深まりへ移る）
✅ <b>タイムライン調整</b>（寝つくころ・深くなるころを感覚的に動かせる。任意）
✅ <b>低刺激の画面</b>（暗い部屋でまぶしくない。就寝中に通知やダイアログを出さない）

<b>【安心して使えます】</b>
マイク・センサー・位置情報は使いません。サーバーを使わず、設定や記録はすべて端末内にのみ保存します。
機内モードでも、寝かしつけ再生・設定・記録の確認ができます。

<b>【料金について】</b>
寝かしつけの基本機能（テンポの変わる再生・自動停止・タイムライン調整・記録）はすべて無料です。
有料は音色の楽しみを広げる音源パックの買い切りのみで、サブスクリプションはありません。
寝つきに直結する機能を有料で解放することはありません。

<b>【ご利用にあたって】</b>
NeteClay は寝かしつけの環境を音で支えるアプリで、医療・治療を目的としたものではありません。
睡眠の測定・診断は行いません。子どもの睡眠そのものを保証するものではありません。
子どもの睡眠や発達について気がかりがある場合は、専門家にご相談ください。
```
