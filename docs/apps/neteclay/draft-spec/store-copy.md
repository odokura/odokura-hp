# NeteClay 外向け文言

ストア、スクリーンショット、広告文、Web 掲載で使う表現の安全ラインを定義する。
仕様本文では説明のために睡眠状態を扱うが、外向けには医学的・治療的な効果を主張しない。

## 基本方針

- 「寝かしつけを支える音楽アプリ」として説明する。
- 「睡眠を改善する」「眠れるようになる」ではなく、「起きにくい環境を支える」「寝かしつけの流れに寄り添う」と表現する。
- マイク・センサーで睡眠を検出するような印象を与えない。
- 子どもの状態を診断・測定する表現を使わない。

## 使ってよい表現

- 抱っこやトントンに合うテンポで始まる。
- 時間とともに音がゆっくり落ち着く。
- 寝ついたあとも低い音で見守る。
- 眠りが深くなるころ、静かにフェードアウトする。
- 生活音が気になりにくい環境を支える。
- マイクやセンサーを使わず、端末内で完結する。
- ホームのワンタップで始められる。

## 避ける表現

- 必ず寝る。
- 夜泣きが治る。
- 睡眠の質が改善する。
- 睡眠障害を改善する。
- 医師監修でないのに医学的効果を示す。
- 睡眠ステージを検出する。
- 子どもの眠りを測定する。
- 科学的に証明された効果がある。

## ストア説明の方向性

- 主語は親に置く。「親が少ない操作で寝かしつけを始められる」ことを伝える。
- 子どもの睡眠そのものを保証しない。
- 端末完結、マイク不使用、音源パック買い切りを分かりやすく伝える。
- 課金訴求は音色の追加に限定する。寝つき機能を有料で解放するように見せない。

## スクリーンショット文言の例

| 画面 | 方向性 |
| --- | --- |
| ホーム | ワンタップで寝かしつけの音を開始 |
| 再生中 | 抱っこ・トントンに合わせて、音がゆっくり落ち着く |
| こだわり設定 | 寝つくころ、深くなるころを感覚的に調整 |
| 音源パック | 好みに合わせて音色を追加 |

## 審査メモに入れる前提

- 医療アプリではない。
- 睡眠測定・睡眠診断は行わない。
- マイク・センサー・サーバーを使わない。
- 有料機能は音源パックの買い切りであり、寝かしつけ再生の基本機能は無料。
