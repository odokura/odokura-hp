# NeteClay コンセプト

## 一言でいうと

子どもの**寝かしつけ**のための音楽アプリ。抱っこ・トントンに合う比較的速めのテンポで始まり、
時間をかけて自然にゆっくりになる。眠りが深くなるまでの時間帯を低い音で支え、生活音でのぶり返しを防ぎ、
そのあと静かに消える。

## 設計の芯: 父親が楽をする

疲れて子を抱いた親が、半分眠りながらでも使えることを最優先にする。この芯に反する仕様は入れない。

- **既定パスはゼロ操作**: 内蔵の既定値のまま、ホームのワンタップで再生が始まり、放置で自動停止する。
  選択を求めない。
- **調整はすべて任意**: タイムライン調整も再生中のタップも、使いたい親だけが使う。
- **急かさない**: 通知・ダイアログを出さない。記録は見たい親が見るだけ。
- **迷わせない**: 画面ごとの主操作は 1 つに絞る。

## 対象ユーザー

- 主対象: 生後まもない乳児〜未就学児を寝かしつける保護者。
- 子どもの寝かしつけに時間がかかって困っている保護者。
- 抱っこ・トントンのリズムに合う音がほしい人。
- 布団に置くと起きてしまう（「背中スイッチ」）のに悩む人。
- 夜間の生活音で子どもが起きるのを防ぎたい人。
- 子どもに関わるデータをクラウドに送りたくない人。

## ODOKURA での位置づけ

Hatsu-go（子どものことばの成長を記録するアプリ）と同じ「子育て支援 × 端末完結」の系列。
サーバーを使わず、子どもに関わるデータを端末外へ出さない。

## 提供価値

- **ワンタップで開始**: 抱っこしながら片手で始められる。
- **リズムに寄り添う**: 比較的速めのテンポで始まり、時間をかけて自然に減速する
  （抱っこ・トントンが眠りとともにゆっくりになるのを模す）。
- **置いても起きにくく**: 眠りが深くなるまでの時間帯を低く安定した音で支え、生活音をマスクして、
  布団へ移した直後の覚醒を防ぐ。
- **自分の子に合わせられる**: 既定値ですぐ始められ、こだわりたいときだけタイムラインで微調整。
  再生中のタップで実際のタイミングにも合わせられる。
- **端末完結**: マイクもサーバーも使わない。子どもの情報を端末外に出さない。

## 主要ユースケース

| ユースケース | 目的 | 優先度 |
| --- | --- | --- |
| 抱っこしながらワンタップ再生 | すぐ寝かしつけを始める | 高 |
| 何も設定せずワンタップで始める | 細かく考えず始める | 高 |
| 布団へ移す前後を音で支える | 移動時の覚醒を防ぐ | 高 |
| 子のリズムに合わせて調整（任意） | 寝つきやすい設定にする | 中 |
| 就寝後に自動でフェード停止 | 放置してよい | 中 |

## 非目標

- マイクによる検出・睡眠測定はしない。タイミングは既定値・タイムライン調整・再生中のタップで親が与える。
- 睡眠の医学的診断・治療はしない。
- 睡眠ステージ（レム/ノンレム）の詳細計測を謳わない。
- クラウド同期・アカウント機能は持たない。
- SNS 共有・ランキングなどのソーシャル機能は持たない。
- 大人自身の入眠を主対象とした設計にはしない（副次的に使えてよいが、寝かしつけを優先する）。
- 複数の子のプロファイル分けはしない（記録は 1 つにまとめる）。
- ストア・外向け文言では、睡眠改善や医学的効果を主張しない。「起きにくい環境を支える」程度に留める。
  仕様本文の説明的表現はこの限りではない。
