# Takaneko 仕様

Takaneko は、作者本人が自宅 PC で使う private な株式一次情報の収集・分析アプリケーション。
上場企業の開示を定期取得し、企業理解に使える企業ファクトへ分けて SQLite と localhost の lab に保存・表示し、
検証済みの企業ファクトを作者本人が管理する非公開（protected）X アカウントへ投稿する。
第三者への配信、公開 Web サイト、外部通知は行わない。リアルタイム株価、出来高、指数などの市場データも扱わない。

## 現在地

- 対象は日本株のみ。収集データは自宅 PC の SQLite に無期限保存する。
- 2026-07 のレビューで、最初に提供する価値を XBRL で機械検証できる財務 3 種
  （決算実績・業績予想の修正・配当）に絞った（→ [スコープと段階計画](./scope.md)）。
- 文章系の種別は将来種別として凍結し、段階ゲートを通過するまで有効化しない。
- 収集・正規化・分析を行う bot（TypeScript）と、結果を作者本人へ表示する localhost 限定の
  ローカル分析サイト（Python 第一候補）の二実行体に分けて運用する（→ [ローカル分析サイト](./local-site.md)）。
- 2026-07 に X 投稿方針を変更した。投稿は作者管理の protected アカウント限定・承認フォロワーなしの
  運用で行い、dry_run ゲートを通過するまで実投稿しない（→ [配信](./distribution.md)、[決定ログ](./decisions.md)）。
