# 共有画像

ホームから作成する共有画像と、設定「共有画像設定」の仕様変更をまとめる。変更は次の 2 点。

## 変更1: プリセットの「カスタム」表記を廃止

### 現状

- プリセット（家庭内共有／SNS向け／専門家向け）は、表示トグルの組み合わせを一括適用するボタン（`app/(tabs)/settings.tsx` の PRESET_CONFIGS）。
- 個別トグルを変更するたびに detectPreset が現在の組み合わせをプリセット定義と照合し、一致すればそのプリセットをハイライト、どれとも一致しなければ「カスタム」ピルを表示する。判定結果は `ShareImageSettings.preset` として保存される。

### 決定

- **「カスタム」表記と、その判定機構（detectPreset・保存される preset フィールド）を廃止する**。組み合わせ状態の逆引き表示は意味が分かりづらく、価値に見合わない。
- プリセットボタンは**押したときに組み合わせを適用するだけの一方向アクション**とする。選択状態のハイライトは持たない（適用後に個別トグルを変えても何も表示は変わらない）。
- `ShareImageSettings` から preset を削除する（SharePreset 型・sharePresetCustom 文言キーも削除）。保存済みデータに残る preset キーは読み込み時に無視されるだけで害はない（マイグレーション不要）。

## 変更2: 音の広がりを共有画像に掲載できるようにする

[ホームの50音ヒートマップ](./kana-heatmap.md)を共有画像にも掲載できるようにする。

### 設定

- `ShareImageSettings` に showKanaHeatmap（boolean）を追加。**既定は false**（既存トグルの既定と同じ最小構成。既存ユーザーの共有画像は変化しない）。
- 共有画像設定に「音の広がりを表示する」トグルを追加する。**ロケールが ja のときのみ表示**（[ホームの多言語方針](./kana-heatmap.md#多言語i18n方針)と同じ。ja 以外ではトグルも掲載も無い）。
- プリセットにも showKanaHeatmap の値を含める:
  - 家庭内共有 = ON（成長の見せどころとして載せる）
  - SNS向け = OFF（画像を長くしない・情報を絞る）
  - 専門家向け = ON（音のインベントリは専門家に有用）

### 共有画像での描画

- 掲載条件: ロケールが ja かつ showKanaHeatmap が ON。
- 位置はグラフセクションの直後・フッター（アプリ名）の前。見出しはグラフと同じ小見出しスタイルで「音の広がり」（kanaHeatmapTitle を共用）。
- 集計はホームと完全に同一（グループ・発語済み判定・集計ソース設定 kanaHeatmapSource を共用）。
- 共有画像は白背景固定のため、**ヒートマップの配色はライトテーマに固定**する（端末のダークテーマに追従しない）。
- 「かなで数えられない N 語は含まれていません」の注記は共有画像には**出さない**（共有相手には文脈がなくノイズになる）。
- サイズ: 共有画像の描画幅（width − 左右 padding 20×2）を calcCellSize に渡す。セル上限 24pt・中央寄せ・拗音の全幅引き伸ばしなどのルールはホームと同じ。共有画像は幅確定で描画されるため onLayout 実測は不要。

### 実装方針

- 現在の `components/kana-heatmap.tsx` はカード・テーマ切替・onLayout 実測を含むため、**グリッド描画部を presentational な共有部品に切り出し**、ホームのカードと共有画像の両方から使う（例: KanaHeatmapGrid に counts・セルサイズ・配色モードを渡す）。集計・セル定義・サイズ計算は `lib/kana-heatmap.ts` をそのまま共用する。

## テスト観点

- preset フィールド削除後も、preset キーを含む既存保存データの読み込みが壊れない（deep merge で無視される）
- showKanaHeatmap の後方互換（保存済み設定にキーが無ければ false になる）
- プリセット適用で showKanaHeatmap が各プリセットの定義値になる
- ロケールが ja 以外のとき、共有画像設定のトグルが表示されず、共有画像にも掲載されない
- showKanaHeatmap が OFF（既定）のとき共有画像に掲載されない
- 掲載時はライト配色固定・注記なし
- ホーム側のヒートマップ表示がグリッド部品の切り出し後も従来どおり（既存テスト green）

## 未確定・論点

- 共有画像内でのセル・文字サイズの見え方は実機キャプチャで微調整
