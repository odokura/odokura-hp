ストア掲載、スクリーンショット、Web 掲載で使う表現の安全ラインと、各ストアの申請項目をまとめる。

## 基本方針

- 「子どもの発語・発達を記録する育児アプリ」として説明する。
- 記録・可視化の価値を伝え、診断・治療・矯正の効果を主張しない。
- 「発達を改善する」「言葉が増える」ではなく、「記録できる」「振り返れる」と表現する。
- 主語は親に置く。親が子どもの様子を残し、振り返り、専門家に説明しやすくなることを伝える。
- データは端末内保存で、外部サーバーにアップロードしないことを明示する。
- 課金訴求は Premium（バックアップ・データ移行・動画無制限）に限定し、記録の基本機能は無料と示す。

## 使ってよい表現

- 語彙の広がりや理解度、発音の変化を記録できる。
- 動画と状況メモを添えて、その場の雰囲気ごと残せる。
- 月ごとのグラフ・サマリーで成長を振り返れる。
- データは端末内にのみ保存する（外部サーバーにアップロードしない）。
- 定期健診や発達支援の場で、子どもの様子を親が説明しやすくなる。

## 避ける表現

- 発達が改善する／発達の遅れが治る。
- 発達障害を診断・判定できる。
- 医師監修でないのに医学的な効果・診断を示す。
- 必ず言葉が増える。
- 発達を測定・評価する（記録であって測定・診断ではない）。

## アプリ名の扱い

App Name / タイトルはストア名・商標の確認済み表記に合わせる。表記を変える場合は全ページを同時にそろえる。
