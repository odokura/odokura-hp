## タイトル

```
Hatsu-go：わが子の視点を理解。発語・発達記録アプリ
```

## 短い説明

```
わが子の視点がもっとわかる。語彙の広がりや発音の変化までシンプルに記録。知育や成長の深まりを実感できる、新しい発達記録アプリです。
```

## 詳細説明

```
<b>【発語を記録して、子供の世界の広がりを見よう】</b>
Hatsu-go（ハツゴー）は、子供の日々の語彙の広がりや、言葉をどれぐらい理解してるかを記録することで、子供の認識の広がり、世界の見え方が変わっていく様子が記録できるアプリです。
赤ちゃんが初めて喋った時の喜びと驚き。どんどん増えていく新しい単語。今しか聞けない面白い言い間違え。そして、はっきりとした「パパ」「ママ」。
Hatsu-go では、単語ごとに動画と時間軸に沿って記録でき、言葉の広がりと共にその場の状況や雰囲気まで、振り返ることができます。

<b>【親が作った発語・成長記録アプリ】</b>
開発者は、発達障害の娘をもつ一人の親です。
そのため、このアプリには語彙ごとに「模倣して発語」や「理解して発語」など、言葉の理解度や発声時の状況を細かく記録できるようにしています。
月ごとにグラフ化・集計されるサマリー機能により、どの言葉をいつ頃から使い始めたか、どんな場面か等が一目でわかります。
もちろん、定型発達のご家庭でも便利にお使い頂けます。定期検診や発達支援の場で、親が子供の様子を正確に説明しやすいよう設計しており、
記録やグラフ化など、育児をサポートする基本機能はすべて無料で無制限にご利用いただけます。

<b>【主要機能】</b>
✅ <b>発語ログ記録</b>（日時、語彙、理解度、状況メモ、動画を記録）
✅ <b>グラフ化・可視化</b>（語彙数や理解度を月ごとに分かりやすくグラフ化）
✅ <b>サマリーと画像保存</b>（語彙の集計結果を画像として保存・共有）
✅ <b>履歴の一覧表示</b>（登録された語彙ごとの推移や履歴をソート検索）
✅ <b>安心バックアップ</b>（Premium機能）
✅ <b>端末間・OS間データ移行</b>（Premium機能）
✅ <b>動画投稿無制限化</b>（Premium機能）

<b>【安心セキュリティ・安心プレミアム】</b>
大切な子供の発語データや動画は、外部サーバーにアップロードせず、使用端末内にのみ安全に保存（ローカル保存）されます。
機種変更などの際にも安心してデータを引き継げるよう、Premium機能にて「暗号化処理によるバックアップとデータ移行」をサポートしています。
<b> Premiam機能は、1度の購入で永続的に利用可能（買い切り）です。</b>

<b>【対象年齢】</b>
・対象年齢：1歳、2歳、3歳、4歳、5歳〜
・年齢に縛られず、それぞれの成長ペースや獲得段階に合わせて長く使えます。

<b>【免責・重要事項】</b>
<b>Hatsu-go</b>は、親が子供の発語発達を記録するための、管理ツールです。
本アプリは、医学的診断、専門的な助言、または治療計画を提供するものではありません。専門家による評価に代わるものではありません。
子供の発語、聴覚、または全体的な発達に関して懸念がある場合は、必ず専門の小児科医や言語聴覚士などの専門家にご相談ください。
```
