## App Name

```
Hatsu-go：發語・發展記錄
```

## Subtitle

```
陪伴孩子詞彙與發音的成長
```

## Promotional Text

```
用影片記錄孩子的第一句「話」的發語・成長 App。詞彙的增加、理解程度與發音變化，都能搭配當下情境備註與影片輕鬆記下。每月圖表與摘要讓你回顧成長軌跡，看見孩子眼中的世界如何一點一點擴展。
```

## Description

```
【記錄發語，看見孩子世界的擴展】
Hatsu-go 是一款透過記錄孩子日常詞彙的增加與語言理解，幫助你更了解孩子如何理解世界、感受世界的 App。
第一次開口說話時的驚喜與感動、不斷增加的新詞彙、只有現在才聽得到的可愛錯音，還有那句清楚喊出的「爸爸」、「媽媽」。
在 Hatsu-go 中，你可以搭配影片，沿著時間軸記錄每一個單字；隨著詞彙慢慢增加，也能一起回顧當時的情境與氛圍。

【由家長親手打造的發語與成長記錄 App】
開發者本身也是一位有發展障礙女兒的家長。
因此，這個 App 也支援以詞彙為單位，細緻記錄像是「模仿後發語」或「理解後發語」等語言理解程度與發聲情境。
透過每月圖表化與統計的摘要功能，可以一眼看出孩子大約從什麼時候開始使用哪些詞語，以及多半出現在什麼情境。
當然，典型發展的家庭也能方便使用。這個 App 的設計，也考慮到定期健檢或發展支援等情境，讓家長更容易準確說明孩子的狀況。記錄、圖表化等支援育兒的基本功能，全部都能免費且不限次數使用。

【主要功能】
發語記錄（記錄日期、詞彙、理解程度、情境備註與影片）
圖表化與可視化（將詞彙數量與理解程度按月整理成清楚圖表）
摘要與圖片保存（可將詞彙統計結果保存為圖片並分享）
歷史記錄清單（可依詞彙查看歷程，支援排序與搜尋）
安心備份（Premium 功能）
跨裝置・跨系統資料移轉（Premium 功能）
影片記錄不限次數（Premium 功能）

【安心安全與 Premium 方案】
孩子珍貴的發語資料與影片，不會上傳到外部伺服器，而是僅安全地保存在你的裝置內（本機保存）。
在更換手機時，你也可以透過 Premium 功能中的「加密備份與資料移轉」，安心延續原本的記錄。
Premium 功能只需購買一次，即可永久使用（買斷制）。

【適用年齡】
・適用年齡：1歲、2歲、3歲、4歲、5歲以上
・不受年齡限制，可依每個孩子自己的成長步調與發展階段長期使用。

【免責聲明與重要事項】
Hatsu-go 是一款協助家長記錄孩子發語發展的管理工具。
這個 App 不提供醫學診斷、專業建議或治療計畫，也不能取代專業人員的評估。
若你對孩子的發語、聽力或整體發展有任何疑慮，請務必諮詢專業的小兒科醫師或語言治療師等專家。
```
