# 課金設計メモ

## 基本方針

- このアプリは、発達の記録として医師や専門家に説明するのに十分な機能を、無料で継続して使えることを重視する。
- 課金要素は、記録の本質機能そのものではなく、容量や利便性、安心感を広げる機能に寄せる。
- つまり `Basic` は「発語記録アプリとして十分使える無料機能」、`Premium` は「記録をもっと豊かに、安心して使うための拡張機能」という位置づけにする。

## Basic で守る価値

- 動画なしの発語記録は無制限に登録可能
- グラフ表示は利用可能
- サマリーのエクスポート機能は利用可能

## Premium で広げる価値

- 動画保存数の制限解除
- バックアップ作成
- バックアップ共有
- バックアップ復元

## 画面上の見せ方の考え方

- `Plan` というより `Premium` / `プレミアム設定` に寄せた表現の方が実態に合う
- `Basic` / `Premium` は単独表示せず、`現在の設定: Basic` のように意味が分かる形で見せる
- 比較表では、Basic を「ほとんど何もできない」ように見せない
- Premium は「本質機能の購入」ではなく「容量・保存・安心の拡張」として見せる
- 買い切りであることを Premium 画面に明示する

## トーン

- 保護者にとって納得感のある線引きにする
- 押し売りに見える訴求は避ける
- 「無料でも十分使える。その上で、より便利に使いたい人向けに Premium がある」という印象を保つ
